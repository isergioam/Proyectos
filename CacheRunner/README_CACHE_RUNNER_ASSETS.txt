Cache Runner - Pack base de assets

Copia la carpeta frontend/public dentro de tu proyecto cache-runner/frontend/public.

Incluye:
- PNGs para fondos, suelo, personaje, obstáculos, coleccionables y UI.
- WAVs reales para salto, impacto, game over, guardado, colección, escudo, doble salto y pausa.
- music.wav como música base corta. Si en el código usas music.mp3, cambia la ruta a /sounds/music.wav o sustituye este archivo por un MP3 real llamado music.mp3.

Rutas principales:
frontend/public/assets/backgrounds/
frontend/public/assets/ground/
frontend/public/assets/player/
frontend/public/assets/obstacles/
frontend/public/assets/collectibles/
frontend/public/assets/ui/
frontend/public/sounds/
